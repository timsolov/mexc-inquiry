# Project for inquiry the MEXC's order book and trades

### Tested with markets:
Binance,
Huobi,
Kucoin,
Mexc,
Kuna,
Exmo,
Okx


### Requirements

- python >= 3.7.0
- pip >= 22.3.1

### Installation

```bash
git clone git@github.com:timsolov/mexc-inquiry.git
cd mexc-inquiry
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### Configuration

see `config.yaml`

### Run

```bash
python3 main.py
```
